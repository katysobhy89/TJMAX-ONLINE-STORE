# TJMaxx Online Store

An online store built using HTML, CSS, JavaScript, and Node.js. The store has features such as cart management, wishlist, checkout, login, and profile management.

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server with `node server/app.js`.
4. Visit `http://localhost:3000` in your browser.
